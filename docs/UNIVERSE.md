# 股票池（UNIVERSE）选择说明

本工程用 `config.UNIVERSE` 自由圈定训练 / 验证 / 测试所用的股票池，取代
原来只有 `USE_CSI300` 的二元开关。

## 语义

```
最终股票池 = (include 各选择器结果取并集) − (exclude 各选择器结果取并集) − 北交所
```

- `include` 为空，或含 `{"type": "all"}`，视为**全市场**（不做 stock_pool 过滤）。
- 北交所（`market == 北交所`）与 ST 股**始终被强制剔除**，与 UNIVERSE 配置无关
  （见 `data_loader.filter_stocks` / `load_basic_info`）。
- 解析后会打印一行 `Universe '{name}': N stocks` 便于核对；若解析出 0 只股票直接报错。
- `UNIVERSE = None` 时回退到旧的 `USE_CSI300` 逻辑，完全向后兼容。

## 选择器类型

| type       | 关键字段 | 含义 | value 可为 |
|------------|----------|------|-----------|
| `index`    | `code`   | 指数成分股 | `"000300.SH"` / `"399006.SZ"` |
| `market`   | `value`  | 板块 | `主板` / `创业板` / `科创板` / `北交所` |
| `industry` | `value`  | 行业（`basic.csv` 的 industry 列） | str 或 list |
| `area`     | `value`  | 地域（`basic.csv` 的 area 列） | str 或 list |
| `codes`    | `value`  | 显式股票代码列表 | list |
| `all`      | —        | 全市场 | — |

`market` / `industry` / `area` / `codes` 的 `value` 既可传单个字符串，也可传列表。

## 各设置下大致股票数

下表为在当前数据（`basic.csv` 共 5522 只上市股票，其中北交所 315 只）上实测的
**解析后股票数**。这是进入加载/特征流程前的候选池规模，最终样本还会再剔除 ST、
新股上市初期（前 `IPO_SKIP_DAYS=40` 个交易日）以及缺数据的行，因此实际进入训练
的股票数会略少。北交所在所有设置中都已剔除。

### 指数类

| 配置 | 选择器 | 股票数 |
|------|--------|--------|
| 沪深300 | `index 000300.SH` | 300 |
| 创业板指 | `index 399006.SZ` | 100 |
| 沪深300 + 创业板指 | 两个 index 并集 | 365 |

> 数据侧 `index_weight/` 目前只有 `000300.SH`（沪深300）和 `399006.SZ`（创业板指）
> 两种成分文件，所以"按指数选股"仅支持这两个。其余自定义池请用 market / industry /
> area / codes 组合。

### 板块类（market）

| 配置 | 选择器 | 股票数 |
|------|--------|--------|
| 主板 | `market 主板` | 3200 |
| 创业板（全部） | `market 创业板` | 1398 |
| 科创板 | `market 科创板` | 609 |
| 主板 + 创业板 | `market [主板, 创业板]` | 4598 |
| 北交所 | `market 北交所` | 0（被强制剔除，会报错） |

### 行业 / 地域 / 显式代码

| 配置 | 选择器 | 股票数 |
|------|--------|--------|
| 银行业 | `industry 银行` | 42 |
| 银行 + 半导体 | `industry [银行, 半导体]` | 235 |
| 深圳 | `area 深圳` | 419 |
| 北京 + 上海 | `area [北京, 上海]` | 901 |
| 指定 3 只 | `codes [...]` | 3 |

> 地域 Top 分布（参考）：浙江 742、江苏 732、北京 485、广东 467、上海 452、
> 深圳 428、山东 315……

### 全市场

| 配置 | 选择器 | 股票数 |
|------|--------|--------|
| 全市场 | `all` 或空 `include` | 约 5207（5522 − 315 北交所，再于加载时剔除 ST） |

### include − exclude 组合示例

| 配置 | 选择器 | 股票数 |
|------|--------|--------|
| 沪深300 去掉银行 | include `index 000300.SH`，exclude `industry 银行` | 276 |
| 主板 去掉上海 | include `market 主板`，exclude `area 上海` | 2936 |
